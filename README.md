## Getting Started

```bash
# Create a new Next.js project using the starter
bunx create-next-app -e https://github.com/ixahmedxi/next-minimal-starter

# Navigate into the project
cd <project-name>

# Install dependencies
bun install

# Run the app
bun dev

# Build the app
bun run build

# Lint
bun check

# Typecheck
bun typecheck
```
